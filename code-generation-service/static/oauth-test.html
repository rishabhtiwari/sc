üìã Instructions

  1. Create GitHub OAuth App: Go to GitHub Settings ‚Üí Developer settings ‚Üí OAuth Apps ‚Üí New OAuth App
  2. Configure App: Use http://localhost:8088/code/auth/github/callback as callback URL

  5. Start OAuth: Click "Start GitHub OAuth" to get the authorization URL
  6. Authorize: Visit the auth URL and authorize the application
  7. Get Session: Copy the session_id from the callback response
  8. Check Status: Verify your authentication status
  9. Connect Repo: Use your session to connect to private repositories
